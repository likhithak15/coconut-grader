# 🥥 Coconut Grader

AI-powered coconut quality classification using YOLOv8 + OpenCV.

## Tech Stack
- **Frontend**: Next.js 14, React, Tailwind CSS
- **Backend**: Flask (Python), OpenCV, YOLOv8
- **Database**: SQLite
